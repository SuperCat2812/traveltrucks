# 🚐 TravelTrucks

TravelTrucks is a modern web application for camper rental built with **Next.js**, **TypeScript**, and **TanStack Query**. Users can browse available campers, filter them by different criteria, view detailed information, and book a camper online.

## 🌐 Live Demo

🔗 https://traveltrucks-gray.vercel.app/

## ✨ Features

- 🏕️ Browse available campers
- 🔍 Filter campers by:
  - Location
  - Vehicle form
  - Engine type
  - Transmission type
- ♾️ Infinite pagination using **useInfiniteQuery**
- 📄 Camper details page
- 🖼️ Swiper image gallery
- ⭐ Customer reviews with a 5-star rating
- 📅 Booking form with validation
- 🔔 Success and error notifications
- ⏳ Loading states
- 📭 Empty state
- ⚠️ Error handling

## 🚀 Getting Started

Clone the repository:

```bash
git clone https://github.com/YOUR_USERNAME/traveltrucks.git
```

Install dependencies:

```bash
npm install
```

Run the development server:

```bash
npm run dev
```

Open:

```
http://localhost:3000
```

Build the project:

```bash
npm run build
```

Run production mode:

```bash
npm start
```
